# Scripts to generate the regressors for hrf convolution of the "Little Prince" project.

1. **texgrid_cn.sh:** 
- Input:  9 TextGrids
- Output: lpp_cn_regressors.csv with 4 columns: word, onset, offset, section number 

2. **freq_cn.sh:**     
- Input:  lpp_cn_regressors.csv generated by textgrid_cn.sh 
- Output: lpp_cn_regressors.csv with 5 columns: word, onset, offset, section number, subtlex_google frequency

3. **pos_cn.sh:**    
- Input: lpp_cn_regressors.csv generated by freq_cn.sh  
- Output: lpp_cn_regressors.csv with 7 columns: word, onset, offset, section number, subtlex_google frequency, POS, content-value,
